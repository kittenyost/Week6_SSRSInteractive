# Week6_SSRSInteractive
Interactive SSRS drill-down reports (Category → Subcategory → Product) with toggle visibility and date-range parameters.
## Files
- SalesByCategoryDrilldown.rdl / SalesByDateRange.rdl
- PDFs: sample exports
- Week6_SSRSInteractive.sln / .rptproj
- ClientWarehouseDS.rds (update connection for your SQL instance)

## Getting Started
Open the solution in Visual Studio with the SSRS extension. Edit **ClientWarehouseDS.rds** to point to your DB, then Preview.
